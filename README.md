# StartPage

A startpage for your browser. it's like a framework that you can customize to your liking.

use it at [start.quntem.co.uk](https://start.quntem.co.uk)

## Features

- Widgets
- Customizable
- shortcuts
- export/import config

## Widgets

- siegeleaderboard
- todo
- calculator
- bitcoin
- weather
- clock
- facebook
- chatgpt
- catphoto
- dictionary
- iframe
- text
- quote
- bookmarks
